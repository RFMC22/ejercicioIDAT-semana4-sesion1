Concepto de Tamaños de Fuentes

  1. Tamaño absoluto

    Este tamaño no depende del contexto del elemento o del entorno de la web.

    pixeles: p { font-size: 16px; }
    puntos: p { font-size: 16pt; }
    milimetros p { font-size: 16mm; }

  2. Tamaño relativo

    Este tamaño si esta relacionado al contexto del elemento o con los tamaños de otros elelemtnos de la web.

    em: p { font-size: 1.2em; } rem: p { font-size: 1.2rem; }
    porcentaje: p { font-size: 20%; }
    unidad basada en la vista p { font-size: 5vw; }
