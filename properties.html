Conceptos de Propiedades de Fuentes

  1. font-family:

    define la familia de la fuente que se aplica al texto.

    body { font-family: "Arial", sans-serif; }

  2. font-size:

    establece el tamaño de la fuente.

    body { font-size: 1.5em; }

  3. font-weight:

    permite dar grosor a la fuente.

    body { font-weight: bold; }

  4. font-style:

    ayuda a definir el estilo de la fuente.

    body { font-style: italic; }

  5. line-height:

    ayuda a brindar altura de linea al bloque de texto.

    body { line-height: 1.5; }

  6. letter-spacing:

    define el espacio entre letras.

    body { letter-spacing: 0.5rem; }

  7. text-align:

    ayuda a establecer una alineación horizontal al texto.

    body { text-align: center; }
